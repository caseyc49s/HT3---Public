# Raw Data

\*__Note that the data is not publicly available__\*
If you do have access please manually load the csv files into this folder in your remote repo.

The files should be named as follows:

- `AlertLevel.csv`
- `EventAppID.csv`
- `EventWarn.csv`
- `Host.csv`
- `RvBasicVehClass.csv`
- `RvBsm.csv`
- `RvZone.csv`
- `Spat.csv`
- `Summary.csv`
